-TEXT TYPING ANIMATION
-QUIZ HAVE DEFAULT 50S TIMER

-CAN CREATE CLASSROOM WITH A CUSTOM PASSWORD

-CAN CREATE QUIZ
-CAN UPLOAD FILE WITH 200MB SIZE
-CAN POST ANNOUNCEMENTS
-CAN MONITOR AND KICK STUDENTS JOINED IN THEIR CLASSROOM
-CAN EDIT AND DELETE UPLOADED QUIZESS
-CAN MONITOR STUDENTS PERFORMANCE AND QUIZ RESULTS

## STUDENTS DASHBOARD

-CAN ANSWER QUIZZES (ONLY ONE TIME)
-CAN SEE ANNOUNCEMENTS
-CAN DOWNLOAD FILES
-CANT COPY AND PASTE AND OTHER KEYBOARD SHORCUTS
